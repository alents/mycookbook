Сделать веб-приложение "Поваренная книга".
Назначение - поиск, добавление, правка рецептов, ведение рейтингов рецептов (кол-во лайков)
В приложении должна быть авторизация, регистрация и разграничение прав. Неавторизованный пользователь
может только искать и просматривать рецепты. Авторизованный пользователь с правами по умолчанию 
может только просматривать, добавлять, удалять и изменять свои рецепты и лайкать чужие.
Существуют роли: 
Админ - может делать все операции, включая назначение ролей пользователям
Менеджер - может делать все операции, кроме назначения ролей другим пользователям
Пользователь(по умолчанию) - лайки, добавление, редактирование, удаление своих рецептов
Ограничения: 
1. нельзя лайкать свои рецепты
2. нельзя лайкать один рецепт более одного раза
 
Поиск должен быть полнотекстовым, то есть искать по словам внутри заголовка и содержимого рецепта.
Должна быть реализована страница с рейтингом рецептов на основе лайков
Рецепты должны делиться на категории, добавлять категории могут только пользователи с ролями Администратор
и менеджер


Рецепт

ид, заголовок, ссылка на список ингредиентов, содержимое рецепта, дата добавления, дата изменения, дата удаления, признак удаления, пользователь

Ингредиент
ид, назание, ссылка на ед.изм.

Еденицы измерения
ид, название

Тэги
ид, название 

Связь тэгов с рецептами
ид тэга, ид рецепта

Лайки
ид рецепта, ид пользователя

Пользователи
ид, логин, пароль

Роли
ид, название

Роли-пользователи
